slack_topics.topics.callables module¶

topics.callables module: plugins for slack-topics.

Below is some of the requirements for plugins.

param None:
returns:
topic for channel with

a message posted to channel. message is typically a link for topic. message and channel can be empty.
rtype:

topic, massage, channel (tuple of strings)
raises:

Not required, but use when needed.
